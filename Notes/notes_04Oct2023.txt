How to use CSS in a HTML file?
	inline
	internal

inline

    <p style="color: red;">This is a paragraph</p>

    <p>This is third para</p>

internal
	style element is used

    <style>
        p{
            color: blue;
        }
    </style>

	we need to create a separate file with extension	css
